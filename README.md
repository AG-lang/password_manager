# 简易密码管理器（Python + Tkinter）

这是一个本地加密的密码管理器，基于 Python 开发，使用 Tkinter 图形界面和 cryptography 加密库。

## 功能特色

- 主密码保护
- 加密数据本地保存
- 随机密码生成
- 记录搜索、复制、备注
- 美观 UI（使用 ttkbootstrap）

## 快速开始

```bash
pip install -r requirements.txt
python main.py
